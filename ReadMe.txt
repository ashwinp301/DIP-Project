STEPS TO RUN THE CODE:
1. Open VS Code.

2. Open the "Code" Folder.

3. Open a new Terminal. Make sure it is open in the current directory.

4. Type the following command: "python app.py"

5. The code will run and you will receive the following message on the terminal:
WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on http://127.0.0.1:5000
Press CTRL+C to quit

6. Hold Left Ctrl and click on the URL to open the webpage.

7. Upload any image that you want to run the weapon detection algorithm on. You will be able to see the image preview on the webpage.

8. Choose any of the 3 available models.

9. Click on "Submit".

10. Once done, you will get the output right beneath the image preview.
